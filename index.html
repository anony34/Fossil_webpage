FoSSIL: A Unified Framework for Continual Semantic Segmentation in 2D and 3D Domains

Continual semantic segmentation is a challenging and underexplored problem in both 2D and 3D domains. The difficulty arises when classes and domains evolve over time, with incremental classes having only a few labeled samples. Models must handle:

  • Catastrophic forgetting of old classes
  • Overfitting on few labeled samples for new classes
  • Domain shifts due to distribution changes

Existing approaches fail to address all these constraints simultaneously. FoSSIL integrates:

1. Guided Noise Injection (GNI)

  • Perturbs overfitted or saturated parameters more strongly
  • Perturbs highly-changing parameters less
  • Preserves critical weights while allowing less critical ones to explore
  • Reduces forgetting, overfitting, and improves robustness

2. Prototype-Guided Pseudo-Label Refinement (PLR)

  • Refines pseudo-labels for incremental classes with unlabeled data
  • Filters out unreliable high-confidence predictions
  • Provides reliable supervision for semi-supervised incremental learning

These components work synergistically to improve stability, generalization, and continual learning across class-incremental (CIL), domain-incremental (DIL), and few-shot scenarios.
